# Umfrage Bachelorarbeit
**Der Einfluss KI-personalisierter Abstimmungswerbung auf die Wahlabsichten in der Schweiz**

Dieses Repository enthält die Web-Umfrage und die Ablagestruktur für die Bachelorarbeit. Die Zielgruppe dieser README ist die Begutachtung der Arbeit.

## Inhalt
- `app/`, `components/`, `lib/`: Quellcode der Umfrage (Next.js).
- `unterlagen/`: Ablage für begleitende Materialien.

## Unterlagen
- `unterlagen/banner/`: Exportierte Banner aus Supabase (Bildmaterial der Umfrage).
- `unterlagen/antworten/`: Exportierte Antworten/Responses aus Supabase.

Die Dateien werden separat hochgeladen.
